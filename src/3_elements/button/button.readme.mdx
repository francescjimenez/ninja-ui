# Button Component

Accessible Button component that enables users to trigger an action or event. It follows the (WAI-ARIA Button Pattern)[https://www.w3.org/TR/wai-aria-practices/#button].

## Import

```js
import { Button } from "ninja-ui";
```

## Usage

```js
<Button></Button>
```
