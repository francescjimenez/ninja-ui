## Usage
#### Install library
```bash
npm install --save ninja-ui
```

#### Theme Provider
Make sure to wrap your app in ThemeProvider to get the default theme:
```javascript

const App = () => <ThemeProvider theme={ThemeBase}>Hey thereðŸ‘‹</ThemeProvider>;
```

#### Importing components from Ninja UI
All components can be imported directly:

```javascript

export default () => (
  <div>
    <Button variant="green" onClick={() => {}}>
      Hello Button
    </Button>
  </div>

```
